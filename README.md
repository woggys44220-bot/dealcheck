# DealCheck

## Lancement

```bash
npm install
npm run dev:all
```

## Configuration API IA

1. Créer un fichier `.env` à la racine du projet.
2. Ajouter votre clé : `OPENAI_API_KEY=...`
3. Lancer `npm run dev:all`

Si la clé API n'est pas configurée, DealCheck continue de fonctionner en mode manuel.
